Detailed Description

An OPENGL implementation of the Game engine. Supporting modern OpenGL this class allows the update and rendering of game objects, sprites and textures. It makes use of simple shaders and manages the creation and destruction of the OpenGL window subsystem. Currently it does not support resizing of windows on the fly, or focus loss.
